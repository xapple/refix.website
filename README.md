## `refix.website` version 0.0.1

`refix.website` is a demo of a website built as a project in a UI/UX course.

See the live version here:

# https://refix.website


## Colors

Interesting links to define the color scheme of the app:

* https://rydmike.com/flexcolorscheme/themesplayground-latest/
* https://material-foundation.github.io/material-theme-builder/
* https://m3.material.io/styles/color/advanced/define-new-colors


## Instructions

To build the app do:

    $ flutter run -d chrome
